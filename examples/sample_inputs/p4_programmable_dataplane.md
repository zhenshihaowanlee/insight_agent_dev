# P4Sketch: Programmable Data Plane Measurement for Congestion Diagnosis

IMC 2026 paper draft.

This paper proposes P4 programmable data plane sketches for network measurement and congestion diagnosis in datacenter fabrics. It targets telemetry visibility rather than direct throughput gain.

The evaluation uses traces, a Tofino testbed, and compares with a baseline telemetry pipeline. It reports p95 detection delay, false positives, memory pressure, ASIC pipeline constraints, operations workflow, and reliability under traffic bursts.
