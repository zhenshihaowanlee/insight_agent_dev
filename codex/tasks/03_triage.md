# Task 03 — Implement triage rules

Goal: classify source rank A/B/C/D and deep-read decision.

Rules:

- A: top conferences, RFC/IETF, standards, production reports.
- B: top labs, big-tech engineering blogs, open-source technical docs.
- C: arXiv, vendor whitepapers, press releases.
- D: secondary summaries, media, no technical details.

Acceptance:

- Unit tests for each source rank.
- Triage result contains rationale.
